#一个verilog实现的数字钟

#属性：
#1、计时，小时数0到23，分和秒为60进制
#2、需要3个按键控制是否时间调整
#3、在调整时间时，相应数码管会闪烁
#4、进入调整状态，计时停止

#方法：
#1、rst：低电平复位
#2、clk：50MHz
#3、set_time_button：按第一下开始进入时间调整模式，第二下退出模式
#4、set_time_change_button：按一下切换调整单位
#5、set_time_add_button：调整时：数字加1；调整分：数字加1；调整秒：归零
